A comprehensive Power BI dashboard that visualizes startup funding distribution, revenue trends over time, industry performance, and country-wise startup insights using dynamic and interactive visuals.
# 📊 Startup Funding & Success Insights Dashboard

## 📌 Overview

This Power BI dashboard analyzes startup funding trends, industry performance, and revenue insights across multiple countries.

## 🚀 Key Features

* 💰 Total funding by industry
* 📈 Revenue trends over time
* 🌍 Country-wise startup distribution
* 🧩 Funding stage analysis (Seed, Series A, B, C, IPO)
* 📊 Industry-wise revenue performance

## 🛠️ Tools Used

* Power BI
* Excel / CSV Dataset

## 📷 Dashboard Preview

![Dashboard](images/dashboard.png)

## 📂 Files Included

* `dashboard.pbix` → Power BI file
* `dataset.xlsx` → Dataset used
* `images/` → Dashboard screenshots

## 🔍 Insights

* Tech & Logistics lead in total funding
* Consistent growth in startup revenue over years
* USA, UK, and China dominate startup count

## 📌 How to Use

1. Download `.pbix` file
2. Open in Power BI Desktop
3. Explore interactive visuals
